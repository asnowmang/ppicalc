#### ppicalc
asnowmang's Fortran 77 pixel density calculator

#### About
This program calculates pixels per unit, inches, centimeters, etc. work. 

#### Example
    % ppicalc
    PIXEL WIDTH: 1440
    PIXEL HEIGHT: 1440
    SCREEN SIZE: 4.5
    PPI: 453
